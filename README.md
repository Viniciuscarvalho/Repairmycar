# Repair My Car

Search for car workshops closer to your location.

# Dependencies

- GoogleMaps;
- KingFisher;
 

# Run tests
#### 1. Run bundle install
`bundle install`
#### 2. Execute fastlane
`bundle exec fastlane test`
#### 3. To generate and open coverage report
`bundle exec fastlane test show_coverage:true`

# Enviroment
- Xcode 9.2
- Swift 4
- Ruby 2.4.0
